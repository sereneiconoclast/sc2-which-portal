# Star Control II: Which portal?
Calculate the best route to a star in Star Control II. This assumes you have
the quasi-space portal spawner.

This classic game from the early 1990's is now available for free as "The Ur-Quan Masters":

https://sc2.sourceforge.net/downloads.php

## Examples

```
which-portal.rb from "025.2 / 145.0" to "175.2 / 145.0"

Travel 150.0 directly through hyperspace to 175.2 / 145.0 (cost: 15.0 units of fuel)

which-portal.rb from "015.2 / 145.0" to "175.2 / 145.0"

Use quasi-space portal at Q506 / 474 (cost: 10.0 units of fuel to use portal spawner)
Emerge in hyperspace at 191.0 / 092.6
Travel 54.7 to 175.2 / 145.0 (cost: 5.5 units of fuel)
Total fuel cost: 15.5 units of fuel
```
